# Mini-Games
A Roblox mini-games framework with 7 games, voting, a shop, powers, and persistent leaderboards.

Generated by [Rojo](https://github.com/rojo-rbx/rojo) 7.6.1.

## Getting Started
To build the place from scratch, use:

```bash
rojo build -o "RPG-Game.rbxlx"
```

Next, open `RPG-Game.rbxlx` in Roblox Studio and start the Rojo server:

```bash
rojo serve
```

For more help, check out [the Rojo documentation](https://rojo.space/docs).
